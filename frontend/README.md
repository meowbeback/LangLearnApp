# Интерфейс

## Как запустить

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер для разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

## Работа с API

По умолчанию приложение использует моковые данные. Чтобы переключиться на работу с реальным бэкендом нужно поменять флаг `USE_REAL_BACKEND = true` в файле `src/services/api.js`.

## Документация

Для генерации документации из комментариев в коде:
```bash
doxygen Doxyfile
```
И открыть `docs/html/index.html` в браузере.
